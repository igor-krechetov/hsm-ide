# Product Context

## Problem Statement
- Designing hierarchical state machines using text-based tools is error-prone and difficult to visualize
- Existing state machine tools lack support for complex hierarchical structures
- Debugging state transitions in code is time-consuming without visual representation

## Solution Approach
- Create a visual IDE specifically for hierarchical state machines
- Implement drag-and-drop interface for state machine elements
- Provide real-time visualization of state transitions
- Include validation and simulation capabilities

## User Experience Goals
- Intuitive interface for creating and connecting state machine elements
- Real-time feedback on state machine validity
- Visual simulation of state transitions
- One-click export to documentation and code formats
- Customizable views for different abstraction levels
